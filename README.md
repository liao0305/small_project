使用说明：

  1、根据requirements.txt安装对应的模块
  
  2、下载微信开发者工具
  
  3、打开微信开发者工具，加载small_program代码，暂时不编译运行
  
  4、后端需要迁移数据库，创建超级用户
  
  5、进入到django后台添加数据
  
  6、启动后端代码，在微信开发者工具中编译运行前端代码
  
  7、具体操作流程详见：https://mp.weixin.qq.com/s/8ZfSmeQQllwTPBaYA7JK7w
